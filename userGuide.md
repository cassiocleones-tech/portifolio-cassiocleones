# Guia do Usuário - Industra Safe Portfolio

## Informações do Site

**Propósito:** Portfólio profissional online para apresentar serviços de segurança industrial e tecnologia, incluindo Gestão Ágil de Projetos, Treinamentos NR-12, Criação de Aplicativos por IA e Inspeções Elétricas.

**Acesso:** Público (não requer login)

---

## Powered by Manus

Este website foi desenvolvido com tecnologias modernas de ponta para garantir performance, segurança e experiência excepcional:

**Frontend:** React 19 com TypeScript oferece uma interface de usuário rápida e responsiva. Tailwind CSS 4 proporciona um design profissional e consistente em todos os dispositivos. Shadcn/ui fornece componentes de interface premium com acessibilidade integrada.

**Deployment:** Infraestrutura de auto-escalonamento com CDN global garante carregamento instantâneo em qualquer lugar do mundo, alta disponibilidade 24/7 e proteção contra picos de tráfego.

---

## Usando Seu Website

O portfólio apresenta suas competências profissionais de forma clara e acessível. A página inicial exibe quatro serviços especializados com descrições detalhadas e recursos principais. Visitantes podem navegar pelo menu superior clicando em "Serviços", "Projetos" ou "Contato" para acessar rapidamente cada seção.

Na seção de projetos anteriores, você encontrará uma área preparada para adicionar seus casos de sucesso reais. Esta seção atualmente mostra exemplos de estrutura que você pode personalizar. Para entrar em contato, os visitantes podem clicar no botão "Chamar no WhatsApp" que abre uma conversa direta com mensagem pré-formatada, ou usar o email exibido.

O site é totalmente responsivo e se adapta automaticamente a smartphones, tablets e computadores, garantindo uma experiência profissional em qualquer dispositivo.

---

## Gerenciando Seu Website

Para visualizar o site ao vivo, clique no botão "View" no card do projeto ou acesse o painel Preview. O painel Code permite baixar todos os arquivos do projeto caso precise fazer backup ou edições avançadas.

Para personalizar o conteúdo do portfólio, especialmente a seção de projetos anteriores, converse com o Manus AI a qualquer momento solicitando as alterações desejadas. Você pode adicionar descrições de projetos reais, incluir resultados alcançados, modificar textos ou ajustar qualquer informação.

Após fazer alterações significativas, é recomendado criar um checkpoint no projeto. Isso permite publicar o site ou reverter para versões anteriores se necessário. O painel Dashboard mostrará estatísticas de visitantes após a publicação.
